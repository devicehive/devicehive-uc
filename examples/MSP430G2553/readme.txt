Simple microcontroller device for DeviceHive example.
It uses DeviceHive binary protocol to connect to the gateway device.
The source code for the sample device is located in MicroProg\src folder.
It should be included in project, afjusted and compiled for a particular microcontroller.
The device it was tested with is PIC 16F88.
A compiled firmware can be located in MicroProg\bin folder.
The device can be locally validated with unit tests from Tests folder.
See the device description and schematics at http://devicehive.com/creating-devicehive-firmware-microcontroller
